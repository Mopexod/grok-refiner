# Grok Refiner

Grok Refiner — это расширение для браузера Chrome, которое добавляет слой управления поверх интерфейсов Grok. Расширение предназначено для пользователей, которые работают с ИИ над текстами, кодом и данными на платформе grok.com и в Grok, интегрированном в x.com.

## Проблема
При работе с Grok часто требуется доработать не весь ответ целиком, а конкретные его фрагменты. Попытки сделать это напрямую в чате приводят к тому, что ИИ меняет то, что менять не просили, теряется контекст обсуждения, приходится вручную копировать текст, засоряя буфер обмена.

## Решение
Grok Refiner предоставляет отдельное рабочее пространство прямо в браузере. Он позволяет изолировать выбранные фрагменты текста, подготовить точные инструкции для каждого фрагмента и не вмешиваться в основной поток диалога с ИИ.

## Основной процесс работы
1. **Выделение текста**: Пользователь выделяет любой текст на странице Grok, рядом с выделением появляется контекстная кнопка «Add Edit».
2. **Сбор фрагментов**: Нажатие кнопки отправляет фрагмент в боковую панель.
3. **Редактирование**: В боковой панели формируется список выбранных фрагментов. Для каждого можно добавить комментарий — что именно нужно изменить, уточнить или переработать.
4. **Вставка в чат**: Кнопка «To Chat» вставляет связку «Цитата + комментарий» в поле ввода чата Grok.
5. **Контроль**: Пользователь самостоятельно отправляет сообщение — расширение не отправляет запросы автоматически.

## Технические особенности
- **Совместимость**: Работает на платформе grok.com и в Grok внутри x.com.
- **Изоляция**: Используется изолированный iframe для стабильной работы без конфликтов с интерфейсом сайта.
- **Сохранность данных**: Выбранные фрагменты и комментарии сохраняются в локальном хранилище браузера, обеспечивая сохранность данных при закрытии вкладки или переключении между сайтами.

## Установка для разработчиков

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/Mopexod/grok-refiner.git
   cd grok-refiner
   ```

2. Установите зависимости:
   ```bash
   npm install
   ```

3. Соберите расширение:
   ```bash
   npm run build
   ```

4. Загрузите расширение в Chrome:
   - Откройте `chrome://extensions/`
   - Включите «Режим разработчика»
   - Нажмите «Загрузить распакованное расширение»
   - Выберите папку проекта `grok-refiner`

## Итог
Grok Refiner — профессиональный инструмент для точечной, контролируемой работы с ответами ИИ. Он устраняет ручную копипасту, снижает количество ошибок и позволяет управлять качеством каждого отдельного фрагмента ответа.
